FAQ

Часто задаваемые вопросы

Кто может зарегистрироваться в системе?

Зарегистрироваться напрямую может только рекламодатель.
Партнёры (арбитражники) регистрируются исключительно по персональной ссылке приглашения, которую создаёт рекламодатель в своём кабинете.

Это сделано специально, чтобы:

рекламодатель контролировал, кто работает с его офферами

исключить мусорный и фродовый трафик

не превращать платформу в открытую CPA-сеть

Как партнёру попасть в систему?

Рекламодатель создаёт ссылку регистрации для партнёров и:

размещает её на своём сайте или лендинге

отправляет партнёру напрямую

Других способов регистрации партнёров нет.

Есть ли отдельные кабинеты для рекламодателя и партнёра?

Да. Это два полностью разных кабинета.

Рекламодатель:

создаёт офферы

управляет партнёрами

настраивает постбеки

смотрит аналитику и финансы

подключает кастомные домены

Партнёр:

видит только доступные ему офферы

генерирует партнёрские ссылки

смотрит свою статистику и доход

не видит других партнёров и данных рекла

Видят ли партнёры бренд платформы?

Нет.
Платформа работает в white-label режиме.

Партнёры видят:

домен рекламодателя

название и оформление рекламодателя

О существовании SaaS-платформы они даже не узнают.

Что включено в бесплатный период?

30 дней полного доступа ко всем функциям выбранного тарифа.

банковская карта не требуется

никаких автосписаний

можно отменить в любой момент

Вы спокойно тестируете систему в боевых условиях.

Есть ли лимиты в бесплатном периоде?

Нет.
Функционал не урезается, ограничения только по тарифам после окончания trial.

Как работает трекинг кликов?

При каждом клике система:

мгновенно генерирует уникальный click_id

определяет GEO, устройство, браузер

сохраняет данные для аналитики и антифрода

Обработка кликов происходит в реальном времени.

Как работает антифрод?

Антифрод включён по умолчанию.

Система анализирует:

IP-адреса (VPN / proxy / hosting)

browser fingerprint

скорость и частоту кликов

подозрительные паттерны поведения

Фродовый трафик автоматически помечается и не учитывается в статистике и выплатах.

Как работают постбеки?

При каждой конверсии система автоматически отправляет postback (HTTP-запрос) на указанный URL.

Поддерживается:

повторная отправка (retry) при ошибках

лог всех отправленных запросов

тестовый режим для проверки

Постбеки работают стабильно и без задержек.

Что если партнёр не использует постбек?

Даже если партнёр не настроил постбек, система всё равно:

фиксирует клики

учитывает конверсии

считает статистику внутри платформы

Если постбек есть — данные передаются и наружу, и остаются у вас.

Есть ли API доступ?

Да. API доступен на тарифах Professional и Enterprise.

Через API можно:

получать статистику

управлять офферами

работать с партнёрами

интегрировать внешние сервисы

Документация открывается после регистрации.

Можно ли подключить свой домен?

Да.
Вы можете подключить любой свой домен для работы с партнёрами.

SSL-сертификат выпускается автоматически (Let’s Encrypt)

настройка занимает 5–10 минут

партнёры работают только через ваш домен

Поддерживаются ли выплаты в криптовалюте?

Да.
Поддерживаются автоматические выплаты через:

Binance, Bybit, Kraken, Coinbase, EXMO, MEXC, OKX.

Доступны холды, история выплат и ручное подтверждение.

Можно ли добавить команду?

Да.
Вы можете добавить менеджеров, аналитиков и финансистов с разными уровнями доступа.

Каждый видит только то, что ему разрешено.

Подходит ли платформа для открытых CPA-сетей?

Нет.
Платформа создана для:

рекламодателей

private / in-house партнёрских программ

агентств и команд

Если вам нужна открытая CPA-сеть с публичной регистрацией — это не наш формат.
